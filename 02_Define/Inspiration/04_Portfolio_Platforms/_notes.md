# Portfolio Platforms — Reference Notes

**What goes here:** Behance, Cargo, Readymag, Are.na, personal portfolio sites (especially designers' own sites, not the platforms). This is primary competition / adjacent space.

**What we're trying to learn:**
- What makes a gallery feel curated vs. cluttered?
- How is work framed — what information appears with each piece?
- What does bad portfolio UI look like? (Capture this too — it's useful for the "what we are not" list.)
- How do the best ones handle the thumbnail-to-detail transition?

**Look for:** How the gallery treats work that doesn't have a polished hero image. That's the state most Closure captures will be in initially.

**What to bring back:** 1 example of a gallery done well, 1 done badly. Note specifically what the bad one does wrong.
