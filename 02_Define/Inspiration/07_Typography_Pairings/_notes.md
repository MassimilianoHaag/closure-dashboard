# Typography Pairings — Reference Notes

**What goes here:** Type specimen screenshots, font pairing examples, app UI using type well. Prioritize pairings actually used in production tools or editorial contexts — not type specimen sites.

**What we're trying to learn:**
- What humanist sans-serifs feel professional without feeling corporate?
- Does Closure need a serif at all — or does the editorial register come from spacing and weight alone?
- What does body type look like at case-study reading length? (300–500 words, mobile-first.)
- How do size and weight alone create hierarchy without needing color?

**Look for:** The pairing that feels like a considered journalist's notebook — not a startup, not a newspaper, somewhere in between.

**What to bring back:** 2–3 pairings with font names noted. Flag one as the candidate to prototype with.
