# Calm Professional Tools — Reference Notes

**What goes here:** Linear, Things, Cron, Pitch, Superhuman, Raycast — tools used by professionals that stay focused without going corporate.

**What we're trying to learn:**
- How do they handle whitespace — is it structural or decorative?
- How do they stay visually quiet while still feeling premium?
- How do they handle secondary information without cluttering primary actions?
- What does "professional restraint" look like vs. "boring enterprise grey"?

**Look for:** The difference between calm and cold. Some tools achieve both; most achieve neither. Find the ones that feel like a focused colleague rather than a filing cabinet.

**What to bring back:** 2–3 layout or spacing treatments. Note what color temperature they use for backgrounds and surfaces.
